=============================
djactasauth
=============================

Django authentication that allows you to login as someone else

Quickstart
----------

Install djactasauth::

    pip install django-act-as-auth

Then use it in a project::

    import djactasauth

Release Notes
-------------

* 0.1.0 - initial release
  * supports Django 1.5 and 1.8 on python 2.7
